BAB 01

Pengantar Pemrograman Komputer INTRO 1

Komponen Komputer

Komputer terdiri dari dua komponen utama:

  • Hardware (Perangkat Keras) — komponen fisik: CPU, RAM, storage, I/O
  • Software (Perangkat Lunak) — program yang berjalan di atas hardware. Terbagi menjadi Sistem Operasi dan Aplikasi.

CPU (Central Processing Unit) adalah otak komputer yang mengeksekusi instruksi. RAM menyimpan data sementara yang sedang diproses.

Bahasa Pemrograman

Ada 3 tingkatan bahasa pemrograman:

  • Bahasa Mesin — kode biner (0 dan 1), langsung dimengerti CPU
  • Bahasa Assembly — menggunakan mnemonik (MOV, ADD, SUB), masih dekat mesin
  • Bahasa Tingkat Tinggi — mudah dibaca manusia: Java, Python, C++

Algoritma & Flowchart

Algoritma adalah urutan langkah-langkah logis untuk memecahkan suatu masalah. Sifat algoritma yang baik: Terbatas, Pasti, Masukan jelas, Keluaran jelas, Efektif.

Contoh algoritma menghitung luas persegi panjang:

Pseudocode
MULAI
  Baca panjang, lebar
  luas ← panjang × lebar
  Tampilkan luas
SELESAI

SistemBasisDigitContoh

BAB 02

Pengenalan Bahasa Java INTRO 1

Sejarah Java

Java diciptakan oleh James Gosling di Sun Microsystems pada tahun 1991 dengan nama awal "Oak". Dirilis secara publik tahun 1995. Prinsip utama: "Write Once, Run Anywhere" (WORA) — kode Java dikompilasi menjadi bytecode yang dapat dijalankan di semua platform yang memiliki JVM.

Fitur-Fitur Java

  • Sederhana — sintaks mirip C++ tetapi menghilangkan fitur kompleks seperti pointer
  • Berorientasi Objek — semua kode ada di dalam class
  • Platform Independent — dikompilasi ke bytecode, dijalankan di JVM
  • Aman — verifikasi bytecode, tidak ada pointer langsung
  • Multithreaded — mendukung eksekusi program secara bersamaan
  • Garbage Collection — memori dikelola otomatis, programmer tidak perlu membebaskan memori manual

Proses Kompilasi & Eksekusi Java

Alur Program Java:
Kode Sumber (.java) → Compiler (javac) → Bytecode (.class) → JVM → Dieksekusi

JVM (Java Virtual Machine) adalah mesin virtual yang mengeksekusi bytecode. Setiap platform (Windows, Linux, Mac) memiliki JVM sendiri, itulah yang membuat Java portabel.

Contoh Operator
public class ContohOperator {
    public static void main(String[] args) {
        int a = 10, b = 3;

        System.out.println(a + b);   // 13
        System.out.println(a - b);   // 7
        System.out.println(a * b);   // 30
        System.out.println(a / b);   // 3 (bukan 3.33!)
        System.out.println(a % b);   // 1

        // Untuk hasil desimal, gunakan double
        double hasil = (double) a / b;
        System.out.println(hasil);    // 3.3333...

        // Pre vs Post increment
        int x = 5;
        System.out.println(x++);  // tampil 5, lalu x jadi 6
        System.out.println(++x);  // x jadi 7 dulu, tampil 7
    }
}

Pengenalan Array

Array adalah variabel yang menyimpan sekumpulan data bertipe sama. Setiap elemen disimpan secara berurutan di memori dan diakses lewat indeks yang dimulai dari 0.

Analogi: Bayangkan array seperti loker bernomor. Loker 0, 1, 2, dst — setiap loker isinya sama jenisnya (semua isi int, atau semua String, dsb).

Membuat Class Sendiri

Mahasiswa.java
public class Mahasiswa {
    // Atribut (field) — biasanya private
    private String nama;
    private String nim;
    private double ipk;

    // Constructor — dipanggil saat new Mahasiswa()
    public Mahasiswa(String nama, String nim, double ipk) {
        this.nama = nama;   // this = merujuk atribut object ini
        this.nim = nim;
        this.ipk = ipk;
    }

    // Constructor kosong (default)
    public Mahasiswa() {
        this("Tidak Diketahui", "000000", 0.0);
    }

    // Getter dan Setter
    public String getNama()  { return nama; }
    public void   setNama(String nama) { this.nama = nama; }

    public String getNim()  { return nim; }
    public double getIpk()  { return ipk; }

    // Method
    public String getPredikat() {
        if (ipk >= 3.5) return "Cumlaude";
        if (ipk >= 3.0) return "Sangat Memuaskan";
        if (ipk >= 2.5) return "Memuaskan";
        return "Cukup";
    }

    // toString: otomatis dipanggil saat print object
    @Override
    public String toString() {
        return "Mahasiswa{" + nama + ", NIM:" + nim + ", IPK:" + ipk + "}";
    }
}

BAB 11

Pewarisan, Polimorfisme, dan Interface INTRO 1

Pewarisan (Inheritance)

Class anak (subclass) mewarisi semua atribut dan method dari class induk (superclass) menggunakan keyword extends.

Pewarisan.java
// Superclass (class induk)
public class Kendaraan {
    protected String merek;
    protected int tahun;

    public Kendaraan(String merek, int tahun) {
        this.merek = merek;
        this.tahun = tahun;
    }

    public void info() {
        System.out.println(merek + " (" + tahun + ")");
    }
}

// Subclass (class anak)
public class Mobil extends Kendaraan {
    private int jumlahPintu;

    public Mobil(String merek, int tahun, int pintu) {
        super(merek, tahun);  // panggil constructor superclass
        this.jumlahPintu = pintu;
    }

    @Override
    public void info() {
        super.info();  // panggil method superclass
        System.out.println("Pintu: " + jumlahPintu);
    }
}

// Penggunaan:
Mobil m = new Mobil("Toyota", 2022, 4);
m.info();
// Output:
// Toyota (2022)
// Pintu: 4

Abstract Class

AbstractClass.java
public abstract class Bentuk {
    protected String warna;

    public Bentuk(String warna) { this.warna = warna; }

    // Abstract method: WAJIB diimplementasi subclass
    public abstract double hitungLuas();
    public abstract double hitungKeliling();

    // Method biasa (boleh diwarisi langsung)
    public void tampilkan() {
        System.out.println("Warna: " + warna + ", Luas: " + hitungLuas());
    }
}

public class Lingkaran extends Bentuk {
    private double radius;

    public Lingkaran(String warna, double radius) {
        super(warna);
        this.radius = radius;
    }

    @Override
    public double hitungLuas() { return Math.PI * radius * radius; }

    @Override
    public double hitungKeliling() { return 2 * Math.PI * radius; }
}

Penanganan Exception INTRO 1

Apa itu Exception?

Exception adalah kejadian tidak terduga yang terjadi saat program berjalan dan mengganggu alur normal program. Contoh: membagi dengan nol, mengakses array di luar batas, konversi String ke int gagal.

Exception UmumPenyebab

public class ContohException {
    public static void main(String[] args) {
        try {
            // Kode yang berpotensi error
            int[] arr = new int[5];
            arr[10] = 99;  // ArrayIndexOutOfBoundsException!

        } catch (ArrayIndexOutOfBoundsException e) {
            // Tangani jenis exception ini
            System.out.println("Error array: " + e.getMessage());

        } catch (Exception e) {
            // Tangani exception lain (lebih umum, ditaruh di bawah)
            System.out.println("Error umum: " + e.getMessage());

        } finally {
            // SELALU dieksekusi, ada error atau tidak
            System.out.println("Selesai (finally selalu jalan)");
        }
    }
}

Algoritma Sorting INTRO 2

Bubble Sort

Algoritma paling sederhana: bandingkan dua elemen bersebelahan, tukar jika perlu. O(n²).

BubbleSort.java
public static void bubbleSort(int[] arr) {
    int n = arr.length;
    for (int i = 0; i < n - 1; i++) {
        for (int j = 0; j < n - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                // tukar
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
}

Selection Sort

SelectionSort.java
public static void selectionSort(int[] arr) {
    int n = arr.length;
    for (int i = 0; i < n - 1; i++) {
        int minIdx = i;
        for (int j = i + 1; j < n; j++) {
            if (arr[j] < arr[minIdx]) minIdx = j;
        }
        int temp = arr[minIdx];
        arr[minIdx] = arr[i];
        arr[i] = temp;
    }
}

import javax.swing.*;
import java.awt.event.*;

public class KalkulatorGUI extends JFrame {
    private JTextField txt1, txt2, txtHasil;
    private JButton btnTambah;

        txt2 = new JTextField(8);

                }
            }
        });

        add(txt1); add(new JLabel("+"));
        add(txt2); add(btnTambah); add(txtHasil);
        setSize(400, 80);
        setVisible(true);
    }
}

Pengenalan Generics INTRO 2

Mengapa Generics?

Generics memungkinkan kita menulis kode yang aman untuk tipe data (type-safe) dan dapat digunakan kembali untuk berbagai tipe. Ini menghindari ClassCastException saat runtime.

Generics.java
// Tanpa Generics (lama, tidak aman)
ArrayList list = new ArrayList();
list.add("Halo");
list.add(123);
String s = (String) list.get(1);  // ClassCastException saat runtime!

// Dengan Generics (aman, error saat compile)
ArrayList<String> list2 = new ArrayList<>();
list2.add("Halo");
// list2.add(123);  // ERROR COMPILE → lebih aman!
String s2 = list2.get(0);  // tidak perlu cast

Membuat Generic Class

GenericClass.java
// T = Type parameter (placeholder)
public class Kotak<T> {
    private T isi;

    public void setIsi(T isi) { this.isi = isi; }
    public T    getIsi()      { return isi; }

    public void tampilkan() {
        System.out.println("Isi kotak: " + isi + " (" + isi.getClass().getSimpleName() + ")");
    }
}

// Penggunaan dengan tipe berbeda
Kotak<String>  kotakTeks   = new Kotak<>();
Kotak<Integer> kotakAngka  = new Kotak<>();
Kotak<Double>  kotakDesimal = new Kotak<>();

kotakTeks.setIsi("Java Generics");
kotakAngka.setIsi(2024);
kotakDesimal.setIsi(3.14);

kotakTeks.tampilkan();    // Isi kotak: Java Generics (String)
kotakAngka.tampilkan();   // Isi kotak: 2024 (Integer)
kotakDesimal.tampilkan(); // Isi kotak: 3.14 (Double)

Generic Method & Wildcard

GenericMethod.java
// Generic method
public static <T extends Comparable<T>> T cariMax(T a, T b) {
    return a.compareTo(b) > 0 ? a : b;
}

// Bisa dipakai untuk int, String, dll
System.out.println(cariMax(10, 25));     // 25
System.out.println(cariMax("Ani", "Budi")); // Budi

// Wildcard (?) — tipe tidak diketahui
public static double totalList(List<? extends Number> list) {
    double sum = 0;
    for (Number n : list) sum += n.doubleValue();
    return sum;
}
// Bisa menerima List<Integer>, List<Double>, dll
